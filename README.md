## Install:

`npm i`
`npx lerna bootstrap`


## Dev mode:

Place .env file in the `server` folder:

```
COCKTAIL_URL=https://www.thecocktaildb.com/api/json/v1
COCKTAIL_API_KEY=1
PORT=3000
```
and then:

`npm run dev`


## Build:

`npm run build`
